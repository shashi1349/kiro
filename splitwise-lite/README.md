# Splitwise-Lite

A Spring Boot REST API for tracking shared group expenses, with an algorithm to
simplify the resulting debts into the minimum number of settlement transactions.

> Status: in active development. Full README arrives in module 8.

Tech: **Java 21, Spring Boot 3.3, Spring Data JPA, Spring Security, JWT, PostgreSQL/H2, Flyway, JUnit 5, Mockito.**
